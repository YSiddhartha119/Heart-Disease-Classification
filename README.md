# Heart-Disease-Classification

This project uses machine learning to predict the presence of heart disease based on clinical parameters. It is built using Python's data science stack to perform Exploratory Data Analysis (EDA) and train classification models.

📌 Project Overview
The goal of this project is to build a binary classification model that can predict whether a patient has heart disease or not given a set of medical attributes (age, sex, chest pain type, cholesterol levels, etc.).

🛠️ Tech Stack
Language: Python

Libraries: * Data Manipulation: Pandas, NumPy

Visualization: Matplotlib, Seaborn

Machine Learning: Scikit-Learn
